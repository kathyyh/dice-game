Welcome to Kathy's Game

Dice game fun for everyone!


Rules

  1. Input the names of both players
  2. Set the goal of the game so that the winner has the highest number or the lowest number
  3. Player one clicks the button to roll the dice. Select which slot you want your value to go
  4. Player two clicks the button to roll the dice. Select which slot you want your value to go
  5. Repeat until all empty number slots are filled
  6. Winner will be declared
